# Экономика дипломного проекта

Легковесное Streamlit-приложение для расчета экономической части дипломного проекта и генерации 6 таблиц в формате LaTeX.

## Локальный запуск через `uv`

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
uv sync
uv run streamlit run app.py
```

После запуска приложение будет доступно по адресу `http://localhost:8501`.

## Что умеет

- вводить работы проекта, оклады и оборудование через `st.data_editor`;
- рассчитывать трудоемкость в часах и днях;
- формировать таблицы по зарплатам, сетевой модели, амортизации и итоговой смете;
- экспортировать готовый LaTeX-код через интерфейс.

## Сборка Docker-образа

```bash
docker build -t diploma-economy .
```

## Запуск контейнера

```bash
docker run --rm -p 8501:8501 diploma-economy
```

После запуска приложение будет доступно по адресу `http://localhost:8501`.
